Powerful Reporting Module (PRM) Documentation

Welcome to the PRM documentation. This system provides integration between Salesforce and Slack for sales reporting and DISC-based sales strategies.

Overview

The PRM system is a Model Context Protocol (MCP) server that provides tools for:

  • Fetching Salesforce opportunities
  • Retrieving upcoming events
  • Getting event credits information
  • Running Salesforce reports
  • Generating DISC-based sales strategies

Available Endpoints

MCP Server

Path: /mcp

Description: Main MCP server endpoint for tool interactions

Static Documentation

Path: /public

Description: This documentation and other static files

Available Tools

  • get_todays_opportunities - Get all opportunities closed today
  • get_yesterdays_opportunities - Get all opportunities closed yesterday
  • get_upcoming_events - Get upcoming events for the next 3 months
  • get_event_credits - Get event credits information
  • get_salesforce_report - Run a Salesforce report by name
  • get_disc_sales_strategy - Get DISC-based sales strategy

Authentication

The API requires authentication using an API key in the x-api-key header.

Environment Variables

  • GPT_MODEL - The OpenAI model to use (default: gpt-4o-mini)
  • OPENAI_API_KEY - OpenAI API key
  • SALESFORCE_USERNAME - Salesforce username
  • SALESFORCE_PASSWORD - Salesforce password
  • SALESFORCE_SECURITY_TOKEN - Salesforce security token
