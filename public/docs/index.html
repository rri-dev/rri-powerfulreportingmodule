PRM API Tool Documentation

Available Tools

get_todays_opportunities

Description: Get all opportunities closed today with their name, stage, and owner information.

Parameters: None

Returns: List of opportunities with details including name, stage, amount, and owner.

get_yesterdays_opportunities

Description: Get all opportunities closed yesterday with their name, stage, and owner information.

Parameters: None

Returns: List of opportunities with details including name, stage, amount, and owner.

get_upcoming_events

Description: Get all upcoming events from Event__c table in the next 3 months.

Parameters: None

Returns: List of events with name, dates, type, and location information.

get_event_credits

Description: Get event credits information from the Event_Credits__c table.

Parameters: None

Returns: List of event credits with account, contact, and credit details.

get_salesforce_report

Description: Run a Salesforce report by name and get the results.

Parameters:
  • report_name (string) - The name of the Salesforce report to run

Returns: Report data with rows, columns, and summary statistics.

get_disc_sales_strategy

Description: Get DISC-based sales strategy recommendations for how a seller should approach a prospect.

Parameters:
  • seller_email (string) - Email address of the seller
  • prospect_email (string) - Email address of the prospect

Returns: Personalized sales strategy based on DISC profiles including communication style, presentation approach, objection handling, and closing techniques.

Example Usage

These tools are designed to be used through the MCP protocol. Connect to the server at /mcp endpoint with proper authentication.

Rate Limiting

The API implements rate limiting to ensure fair usage. Default limits are 5 requests per hour per IP address.
